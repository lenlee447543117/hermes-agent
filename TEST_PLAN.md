# AutoGLM-Phone 测试执行计划

- 设备: 三星 Galaxy S23 Ultra (SM-S9180)
- Android 版本: 16
- 测试日期: 2026-04-24
- 应用状态: 已安装，辅助服务已启用

## 测试执行步骤

### Phase 1: TC-01 权限预检测试
1. [ ] 验证应用启动时权限检查功能
2. [ ] 确认暖阳橙风格的引导弹窗正常显示
3. [ ] 测试权限跳转功能正常

### Phase 2: TC-02 链路完整性测试
1. [ ] 点击"按住说话"

2. [ ] 说出"下载排在第一个的应用"
3. [ ] 验证 AutoGLM 正确识别坐标
4. [ ] 验证辅助服务精准点击下载按钮

### Phase 4: TC-04 复杂意图流测试
1. [ ] 说出"去淘宝帮我搜一下新款手机"
2. [ ] 验证：点击桌面淘宝
3. [ ] 验证：点击搜索框

### Phase 5: TC-05 App 切换测试
1. [ ] 在微信界面说"返回桌面"
2. [ ] 验证辅助服务调用 GLOBAL_ACTION_HOME
3. [ ] 验证手机回到桌面
